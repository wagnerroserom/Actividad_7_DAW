Este proyecta una vez analizado implementa una "Agenda de Contactos" usando Flask (Python) y MySQL de Xampp de las indicaciones de la práctica guiada que se proporciona.
El sistema permite: Registrarse como usuario, Iniciar y cerrar sesión, Agregar, editar y eliminar contactos, mostrar unicamente los contactos del usuario autenticado y usar paltillas con Jinja2 y Bootstrap.
Requisitos previos:
- Python 3.10 o superior  
- XAMPP con MySQL 
- VSCode  
- Navegador web  
- Paquetes instalados con `pip`
El sistema utiliza flask-session
Hash seguro de contraseñas con werkzeug.security
